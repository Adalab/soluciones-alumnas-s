Lenguajes de programación

HTML, CSS y JavaScript son los tres lenguajes que están en el centro de crear aplicaciones web.

¿Qué es HTML?

El lenguaje HTML es el lenguaje de marcado de hipertexto. Es el lenguaje donde se define la información o el contenido del documento. Estructura la información.

¿Qué es CSS?

El lenguaje CSS es el cascading style sheets. Es el lenguaje donde se especifica el diseño del documento, maneja todo lo relacionado con la parte visual. Es el diseño de la información.

¿Qué es JavaScript?

El lenguaje JavaScript es el lenguaje que hace que todo sea interactivo. Es realmente el lenguaje de programación que nos permite crear sitios web. Es la interactividad de la web con la información.
